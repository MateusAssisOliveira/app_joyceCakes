# Pasta `styles/` – Temas e cores globais

Aqui ficam os **temas do app** (cores, tokens visuais).

- **`themes.css`** — Único lugar onde as cores são definidas. Cada bloco `.theme-xxx` é um tema completo (cards, botões, sidebar, gráficos).

Para mudar cores: edite **`themes.css`**.  
Para trocar qual tema está ativo: edite **`src/lib/theme.ts`** → `activeThemeClass`.

Detalhes: **docs/TEMAS.md**.
